<img width="758" height="435" alt="image" src="https://github.com/user-attachments/assets/978ae6a9-03d6-4e34-8b02-e28700c318d0" />

_**Выводы:**_

- Среднее количество попыток примерно одинаково коррелирует и с максимальным, и с минимальным.

- Есть только одна пара признаков со слабой корреляцией (min-max).

- Если считать корреляцию более 60% сильной, то есть 2 пары признаков с сильной корреляцией (mean-min, mean-max).

- Нет признаков с 100% корреляцией.

- Нет признаков с сильной обратной зависимостью.
